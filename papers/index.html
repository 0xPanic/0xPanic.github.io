Identifying Previously Installed Apps on Android Devices

Creating a process to determine if an app was previously installed on an android device. This process is based on identifying the files created by the app at different stages and figuring out which files are unique to the app and identical across installs of the app.

An Evaluation of FOSS Static Code Analysis Tools for C

We tested out three static code analysis tools that are designed for finding vulnerabilites in C code. We ran them on NIST's Juliet test case suite to see how effective they were at identifying vulnerable code for each of the CWEs in the C programming language.
